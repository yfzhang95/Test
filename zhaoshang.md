3. 现有n瓶化学试剂，第i瓶有ai升的试剂，小招喵的目的是是每一种化学试剂的量相同。可以进行以下两种操作：
    + 选择一种试剂，将他的试剂量翻倍，即变为2ai升试剂  
    + 选择一种试剂，将他的试剂量减半，即变为ai/2升试剂  
我们假定每个瓶子的容量是无穷大的，问最少需要进行几次操作才可以使瓶中的试剂量相同。
+ 输入描述：
    >第一行是一个整数n(1 <= n <= 10^5)  
    >第二行包含n个整数ai(1 <= ai <= 10^5>>)
+ 输出描述
    >一个整数，表示问题的答案。
+ 示例:  
    输入：  
        >3  
        >4 8 2  
    输出：  
        > 2
